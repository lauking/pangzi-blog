GOLang连接mysql

发表于 12月 17 2013

昨天想着写下go连接mysql数据，以为使用
database/sql
这个包就可以连接成功，后来在cnblog上查了下，需要使用到
github.com/go-sql-driver/mysql
这个驱动包。好心人都会写明在哪怎么下载这个包，运行:
go get github.com/go-sql-driver/mysql

会下载到你的GOPATH路径的src 下

实例代码如下：

package main
import (
"database/sql"
"fmt"
_ "github.com/go-sql-driver/mysql"
)
func testSelect() {
    db, err := sql.Open("mysql", "root:111111@tcp(127.0.0.1:3333)/showme?charset=utf8")
    if err != nil {
      fmt.Printf("connect err")
    }
   rows, err1 := db.Query("select userid,username from tb_user limit 0,5")
   if err1 != nil {
      fmt.Println(err1.Error())
      return
   }
   defer rows.Close()
   fmt.Println("")
   cols, _ := rows.Columns()
   for i := range cols {
      fmt.Print(cols[i])
      fmt.Print("\t")
   }
   fmt.Println("")
   var userid int
   var username string
    for rows.Next() {
    if err := rows.Scan(&amp;userid, &amp;username); err == nil {
        fmt.Print(userid)
        fmt.Print("\t")
        fmt.Print(username)
        fmt.Print("\t\r\n")
    }
}
}
func main() {
   testSelect()
}


总结:多看文档，多动手谢谢！

    eclipse下Go的开发配置

    发表于 12月 7 2013
    1、进入help->Install New Software 在弹出框中
    2、选择Add
    3、在name中输入go plugin，在location中输入：http://goclipse.googlecode.com/svn/trunk/goclipse-update-site/*，OK退出。
    4、等待一会，出现GoClipse，选择后Next
    5、引导安装完成后，配置Go的preference
    6、配置完成后，就可以建立Go工程，编写go代码了
